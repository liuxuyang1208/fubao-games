# 📍 GitHub Release 创建完整指南

## 🔍 **第一步：访问您的GitHub仓库**

```
https://github.com/您的用户名/fubao-games
```

## 🏷️ **第二步：找到Releases页面**

### **方法一：顶部导航栏**
```
仓库主页顶部：
┌─────────────────────────────────────────────────┐
│ <Code> <Issues> <Pull requests> <Actions> <Projects> <Security> <Insights> │
│         ↑                                       │
│         └── 点击这里或旁边的"Releases"             │
└─────────────────────────────────────────────────┘
```

### **方法二：右侧快速链接**
```
仓库主页右侧：
┌─────────────────┐
│ About           │
│ Releases (0)    │ ← 点击这里
│ Packages        │
│ Languages       │
└─────────────────┘
```

## 🚀 **第三步：创建新Release**

### **点击创建按钮**
```
在Releases页面顶部：
┌─────────────────────────────────────────┐
│  Create a new release                    │ ← 点击这个绿色按钮
└─────────────────────────────────────────┘
```

### **填写Release信息**
```
Tag version: v1.0.0
Target: main
Release title: 逃离鸭科夫 v1.0.0

描述内容：
🎮 逃离鸭科夫 - 完整游戏版本 v1.0.0

📋 游戏信息：
- 类型：冒险解谜游戏
- 文件大小：943MB
- 支持系统：Windows 10/11 64位
```

## 📁 **第四步：上传游戏文件**

### **Assets区域**
```
在页面底部找到：
┌─────────────────────────────────────────┐
│ Attach binaries                          │ ← 点击这里
└─────────────────────────────────────────┘
```

### **选择文件**
1. **点击"Attach binaries"**
2. **选择您的 escape-duckov.zip 文件**
3. **等待上传完成**（可能需要几分钟）

## ✅ **第五步：发布Release**

```
页面底部：
┌─────────────────────────────────────────┐
│  Publish release                         │ ← 点击这个绿色按钮
└─────────────────────────────────────────┘
```

## 🎯 **常见问题解决**

### **问题1：找不到Releases标签**
**解决方案：**
- 确保您有仓库的写入权限
- 尝试刷新页面
- 检查是否是私有仓库

### **问题2：没有"Create a new release"按钮**
**解决方案：**
- 确保您是仓库的所有者或协作者
- 检查仓库设置中的权限

### **问题3：文件上传失败**
**解决方案：**
- 文件大小不能超过2GB
- 检查网络连接
- 尝试刷新页面重新上传
