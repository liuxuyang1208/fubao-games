## 🌐 部署说明

### GitHub Pages 部署

1. 将代码推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择源分支和文件夹
4. 访问生成的网站链接

### 游戏文件上传

游戏文件通过GitHub Releases分发：

1. 创建新的Release版本
2. 上传游戏文件到Assets
3. 更新网站中的下载链接
4. 用户可直接从GitHub下载
